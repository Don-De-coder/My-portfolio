# Portfolio 2025

Personal site built with Tailwind CSS. Wanted something cleaner than the old bootstrap setup.

## Stack

- Tailwind CSS (CDN)
- Lucide icons
- Vanilla JS

## Structure

- Single page, scroll sections
- Typewriter effect on hero
- Skill bars animate on scroll
- Mobile menu

## Notes

Images are placeholder URLs - swap for local files before deploying. Typewriter words can be edited in the JS at the bottom.
